Game Breakers (Start: {{ game_breaker_start }})
At any given time, we track the number of days since the last game was acquired (currently ago, as noted above). Consider, then, the following: what is the highest number that Last Game Acquired has ever been?

By definition, said highest number must either be today, or a day in the past in which one or more games arrived (on a day in the past where no game arrived, the following day is one higher) - at which point the value would reset to 0.

From here, we may define another emergent value: because non-today values of highest ever Last Game Acquired will inevitably reset to 0 by the arrival of new games, what games arrived to cause that reset and on what day did they arrive?

Game Breakers track those games and days: starting from {{ game_breaker_start }}, mark the date that we went from the highest ever Last Game Acquired to 0, along with the game(s) which appeared that day.
